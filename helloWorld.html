Au cours de cette formation, nous allos voir deux matières :

L'intégration

Le HTML:

Le HTML sert à créer le contenu du site : elle va contenir uiquement la partie textuelle du site.

Le CSS:

Le CSS va nous permettre d'ajouter la couche graphique à notre site : les couleurs, les formes, le fond, etc.

Le développement :

Le Javascript :

Javascript est un langage de programmation de script employé pour renre une page web interactive, le comportement de la page et son appparence pourra varier en fonction de certains paramètres.

Le PHP:

Le php est un langage de programmation libre, utiisé pour générer des pages web dynamiques, qui sont modifiés en fonction de certaines informations (heure, nom de l'utilsateur, formulaire rempli par l'utilisateur etc).

MySQL:

MySQL est un système de gestion de base de données, il permet d'enregistrer des informations remplies par l'utilisateur (exemple: forulaire de création de compte) pour ensuite être réutilisé au moment appropié (ex: connexion à un site internet avec un identifiant et un mot de passe).
